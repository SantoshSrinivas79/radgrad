isAuthorized() → {boolean}

Source:
  • ui/layouts/shared/if-authorized.js, line 31

Determine if the user is authorized to view the current page.
If current page is the landing page, they are authorized.
If current user is an admin, they can see any page.
If current user is an advisor, they can see any student, alumni, or advisor page.
If current user is a student, they can see only their own page (student//).
If current user is a mentor, they can see only their own page (mentor/).
If current user is an alumni, they can see only their own page (alumni/).

Returns:

True if there is a logged in user and they are authorized to visit the page.

Type
boolean
